# Atelier13_simulation_USSD_OM
Simulation du service OrangeMoney en ligne de commande avec Python
## Ce programme permet de :
- Consulter le solde 
- Acheter du credit
- Acheter un forfait 
- Effectuer un transfert 
- Annuler le dernier transfert
- Consulter l'historique
- Sauvegarde automatique du solde et des transferts a l'aide de fichiers json

## Technologies Utilisees 
- Python 3
- JSON pour le stockage de donnees
- Programmation procedurale

## Structure
- main.py
- data_solde.json
- data_transfert.json
